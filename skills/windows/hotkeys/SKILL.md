# Windows Hotkeys Reference

## Modifier Keys

- Use `ctrl` as the primary modifier key (NOT `cmd`)
- Use `win` for Windows system shortcuts
- Common mistake: `cmd+c` should be `ctrl c`

## System Hotkeys

| Shortcut | Action |
|----------|--------|
| `win` | Open Start menu |
| `win s` | Windows Search |
| `win e` | Open File Explorer |
| `win d` | Show desktop |
| `win m` | Minimize all windows (WARNING: system-level, cannot be overridden) |
| `win l` | Lock screen |
| `win i` | Open Settings |
| `win n` | Open Notification Center |
| `alt tab` | Switch applications |
| `alt f4` | Close current application |

## Text Editing

| Shortcut | Action |
|----------|--------|
| `ctrl c` | Copy |
| `ctrl v` | Paste |
| `ctrl x` | Cut |
| `ctrl a` | Select all |
| `ctrl z` | Undo |
| `ctrl y` | Redo |
| `ctrl f` | Find |
| `f3` / `ctrl g` | Find next |
| `backspace` | Delete before cursor |
| `delete` | Delete after cursor |
| `ctrl backspace` | Delete word before cursor |
| `ctrl delete` | Delete word after cursor |

## Text Selection

| Shortcut | Action |
|----------|--------|
| `ctrl shift left` | Select word left |
| `ctrl shift right` | Select word right |
| `shift home` | Select to line start |
| `shift end` | Select to line end |
| `ctrl shift home` | Select to document start |
| `ctrl shift end` | Select to document end |

## Cursor Movement

| Shortcut | Action |
|----------|--------|
| `ctrl left` | Move cursor by word left |
| `ctrl right` | Move cursor by word right |
| `home` | Move to line start |
| `end` | Move to line end |
| `ctrl home` | Move to document start |
| `ctrl end` | Move to document end |

## Browser/Tabs

| Shortcut | Action |
|----------|--------|
| `ctrl t` | New tab |
| `ctrl w` | Close tab |
| `ctrl shift t` | Restore closed tab |
| `ctrl l` / `f6` | Focus address bar |
| `ctrl r` / `f5` | Refresh |
| `ctrl shift r` / `ctrl f5` | Force refresh |
| `alt left` | Back |
| `alt right` | Forward |
| `ctrl 1-9` | Switch to tab N |

## Quick Text Operations

| Goal | Sequence |
|------|----------|
| Clear all text | `ctrl a` then `backspace` |
| Replace all text | `ctrl a` then type new text |
| Copy all | `ctrl a` then `ctrl c` |
