# Microsoft PowerPoint Skill (macOS)

## 应用识别

- 应用名称：Microsoft PowerPoint
- macOS 进程名：Microsoft PowerPoint
- 支持平台：macOS（其他平台暂不支持）

## 启动与创建文档

### 从启动界面创建新文档

**重要**：在 PowerPoint 启动界面，点击"空白演示文稿"时需要 **双击** 才能创建新文档，单击只会选中但不会打开。

```
操作序列：
left_double [空白演示文稿] → wait 1500
```

### 使用 Spotlight 启动

```
hotkey "cmd space" → wait 300 → type "Microsoft PowerPoint" → wait 500 → hotkey "enter" → wait 2000
```

## 常用操作

### 新建幻灯片

**警告**：`Cmd + M` 是 macOS 系统快捷键（最小化窗口），不能用于新建幻灯片！

正确方法：
1. 点击功能区 "开始" 选项卡 → 点击 "新建幻灯片" 按钮
2. 或在左侧幻灯片缩略图区域右键 → 选择 "新建幻灯片"
3. 或使用快捷键 `Cmd + Shift + N`

**注意**：新建的幻灯片会包含预设的文本框（如"单击此处添加标题"、"单击此处添加副标题"）。如果不需要这些预设文本框，应当先选中并删除它们，否则会影响画面布局。

### 选择元素

**重要**：对于文本框，直接单击会进入**文本编辑模式**而不是选中元素！要选中文本框本身（用于移动、删除等操作），必须使用以下方法：

- **Cmd + 单击**：选中元素本身（推荐！对文本框最可靠）。使用 `click` 工具的 `modifiers` 参数：`click([x, y], modifiers=["cmd"])`
- **单击边框**：点击文本框的边框而非内部（不推荐，难以精确定位）
- **Cmd + A**：全选当前幻灯片上的所有元素
- **拖动选框**：使用 `drag` 工具在空白区域拖动，框选多个元素（推荐用于批量选择）
- **点击空白区域**：取消所有选择

**选中 vs 编辑模式**：
- 单击文本框内部 → 进入编辑模式（光标闪烁，可输入文字）
- Cmd + 单击 → 选中模式（显示选择框，可删除/移动）

### 删除元素

1. 先选中要删除的元素（单击或多选）
2. 按 `Delete` 键删除
3. 或右键 → 选择 "删除" / "剪切"

**批量删除**：
- `Cmd + A` 全选 → `Delete` 删除所有元素
- 框选多个元素 → `Delete` 删除选中的元素

**删除预设文本框**：
新建幻灯片后，如果不需要预设的标题/副标题文本框：
1. 在空白区域拖动鼠标框选所有预设文本框（最快方式）
2. 按 `Delete` 一次性删除所有选中元素
3. 或者单击选中单个文本框 → `Delete` 逐个删除

### 插入形状

1. 点击窗口内的 "插入" 选项卡
2. 点击 "形状" 按钮
3. 选择形状类型
4. 在幻灯片上 **拖动** 绘制（不是单击）

### 插入文本框

1. 点击 "插入" 选项卡
2. 点击 "文本框" 按钮
3. 在幻灯片上拖动绘制文本框
4. 输入文字

### 编辑文本

1. **双击** 文本框进入编辑模式
2. 输入或修改文本
3. 点击文本框外部退出编辑模式

## 快捷键

可用快捷键：
- `Cmd + N` → 新建演示文稿
- `Cmd + O` → 打开文件
- `Cmd + S` → 保存
- `Cmd + Shift + S` → 另存为
- `Cmd + W` → 关闭当前窗口
- `Cmd + Shift + N` → 新建幻灯片
- `Cmd + Z` → 撤销
- `Cmd + Shift + Z` → 重做
- `Cmd + A` → 全选
- `Cmd + C/V/X` → 复制/粘贴/剪切
- `Delete` → 删除选中对象

**禁用快捷键**（会被 macOS 系统拦截）：
- `Cmd + M` → 最小化窗口（不是新建幻灯片！）
- `Cmd + H` → 隐藏窗口
- `Cmd + Q` → 退出应用

## 常见问题

### 点击没反应
1. 检查是否点击了正确的位置（功能区 vs 系统菜单栏）
2. 某些按钮需要双击
3. 确保 PowerPoint 窗口处于激活状态

### 形状无法绘制
1. 确保已选择形状类型
2. 需要在幻灯片上 **拖动** 绘制，不是单击

### 窗口被最小化了
如果误按 `Cmd + M` 导致窗口最小化：
1. 点击 Dock 栏中的 PowerPoint 图标恢复窗口
2. 或使用 `Cmd + Tab` 切换到 PowerPoint
