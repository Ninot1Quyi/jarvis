# Microsoft PowerPoint Skill

## 应用识别

- 应用名称：Microsoft PowerPoint
- macOS 进程名：Microsoft PowerPoint
- Windows 进程名：POWERPNT.EXE

## 平台特定指南

# PowerPoint macOS 操作指南

## 启动与创建文档

### 从启动界面创建新文档

**重要**：在 PowerPoint 启动界面，点击"空白演示文稿"时需要 **双击** 才能创建新文档，单击只会选中但不会打开。

```
操作序列：
left_double [空白演示文稿] → wait 1500
```

### 使用 Spotlight 启动

```
hotkey "cmd space" → type "Microsoft PowerPoint" → enter → wait 2000
```

## 界面特点

### 功能区位置

macOS 版 PowerPoint 的功能区（开始、插入、绘图等选项卡）位于：
- 窗口标题栏下方
- **不是** 屏幕最顶部的 macOS 系统菜单栏

注意区分：
- 屏幕顶部 (y ≈ 0-25)：macOS 系统菜单栏（PowerPoint、文件、编辑、视图...）
- 窗口内部：PowerPoint 功能区选项卡（开始、插入、绘图、设计...）

### 插入形状

1. 点击窗口内的 "插入" 选项卡（不是顶部菜单栏的"插入"）
2. 点击 "形状" 按钮
3. 选择形状类型
4. 在幻灯片上拖动绘制

## macOS 特定快捷键

- `Cmd + N` → 新建演示文稿
- `Cmd + O` → 打开文件
- `Cmd + S` → 保存
- `Cmd + Shift + S` → 另存为
- `Cmd + W` → 关闭当前窗口
- `Cmd + Q` → 退出 PowerPoint
- `Cmd + M` → 新建幻灯片
- `Cmd + Z` → 撤销
- `Cmd + Shift + Z` → 重做

## 常见问题

### 点击没反应

1. 检查是否点击了正确的位置（功能区 vs 系统菜单栏）
2. 某些按钮需要双击
3. 确保 PowerPoint 窗口处于激活状态

### 形状无法绘制

1. 确保已选择形状类型
2. 需要在幻灯片上 **拖动** 绘制，不是单击


## 通用操作

### 界面布局

PowerPoint 主要分为以下区域：

```
┌─────────────────────────────────────────────────────┐
│  [文件] [开始] [插入] [绘图] [设计] [切换] [动画]    │  ← 功能区选项卡
├─────────────────────────────────────────────────────┤
│  功能区工具栏（根据选中的选项卡显示不同工具）          │
├──────────┬──────────────────────────────────────────┤
│          │                                          │
│  幻灯片   │           幻灯片编辑区                    │
│  缩略图   │                                          │
│          │                                          │
│          │                                          │
└──────────┴──────────────────────────────────────────┘
```

### 常用快捷键

- `Cmd/Ctrl + N` → 新建演示文稿
- `Cmd/Ctrl + S` → 保存
- `Cmd/Ctrl + Z` → 撤销
- `Cmd/Ctrl + Y` → 重做
- `Cmd/Ctrl + C/V/X` → 复制/粘贴/剪切
- `Cmd/Ctrl + A` → 全选
- `Cmd/Ctrl + M` → 新建幻灯片
- `Delete` → 删除选中对象

### 插入形状

1. 点击功能区的 "插入" 选项卡
2. 点击 "形状" 按钮
3. 从下拉菜单中选择需要的形状
4. 在幻灯片上拖动绘制形状

### 编辑文本

1. 双击文本框进入编辑模式
2. 输入或修改文本
3. 点击文本框外部退出编辑模式

### 添加新幻灯片

1. 点击功能区的 "开始" 选项卡
2. 点击 "新建幻灯片" 按钮
3. 或使用快捷键 `Cmd/Ctrl + M`

## 注意事项

1. **功能区位置**：功能区选项卡（开始、插入、绘图等）位于应用窗口内部，不是 macOS/Windows 系统菜单栏
2. **双击 vs 单击**：某些操作需要双击（如进入文本编辑模式、从启动界面创建文档）
3. **形状绘制**：选择形状后需要在幻灯片上拖动才能绘制，单击只会创建默认大小的形状
