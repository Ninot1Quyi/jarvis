# PowerPoint Windows 操作指南

## 启动与创建文档

### 从启动界面创建新文档

在 PowerPoint 启动界面，点击"空白演示文稿"创建新文档。

```
操作序列：
click [空白演示文稿] → wait 1500
```

### 使用开始菜单启动

```
hotkey "win" → type "PowerPoint" → enter → wait 2000
```

## Windows 特定快捷键

- `Ctrl + N` → 新建演示文稿
- `Ctrl + O` → 打开文件
- `Ctrl + S` → 保存
- `F12` → 另存为
- `Alt + F4` → 退出 PowerPoint
- `Ctrl + M` → 新建幻灯片
- `Ctrl + Z` → 撤销
- `Ctrl + Y` → 重做

## 注意事项

- Windows 版功能区布局与 macOS 版类似
- 使用 `Ctrl` 代替 `Cmd` 作为主要修饰键
